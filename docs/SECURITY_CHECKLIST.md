# Checklist de seguridad

- [x] JWT firmado con clave externa.
- [x] BCrypt para contraseñas.
- [x] Roles: ANALISTA, SUPERVISOR, GERENTE, ADMINISTRADOR.
- [x] Analista opera solo sus tickets.
- [x] Dashboard/reportes restringidos por rol.
- [x] CORS restringido por variable.
- [x] Bean Validation en DTO.
- [x] Errores sin stack trace ni SQL.
- [x] Logs sin contraseñas, tokens ni observaciones completas.
- [x] Excel con protección contra inyección de fórmulas.
- [x] Consultas JPA parametrizadas.
- [x] Credenciales por variables de entorno.
- [x] Demo data desactivable para producción.
